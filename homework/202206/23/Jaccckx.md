# C++: 描述内存分配方式以及它们的区别。

1. 内存分配的区域
   - 栈区
   - 堆区
   - 自由存储区
   - 全局/静态存储区
   - 常量存储区

2. 

- 栈区

  - 由高地址向低地址增长

  - 栈一般用来存放临时变量

  - 由系统管理，与函数调用相关

  - 静态分配由编译器完成，动态分配使用alloc函数

    ```
    char* str = (char*)alloca(length * sizeof(char));
    ```

- 堆区

  - 由new申请，delete删除，容易造成内存泄露
  - 在程序运行结束后，由系统自动回收

- 自由存储区
  - 由malloc申请，free释放
  - 与堆区相似

- 全局/静态存储区
  - 即BSS/DATA段。
  - 不过在c++中他们占用同一片内存区域

- 常量存储区
  - 例如const
  - 或者一些字符串常量
