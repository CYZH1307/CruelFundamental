# 进程和线程
- https://blog.csdn.net/wujiafei_njgcxy/article/details/77098977
- 进程是系统资源分配和调度的基本单位
- 线程是进程中执行运算的最小单位

# 对比
| 特征     | 进程       | 进程     |
|----------|------------|----------|
| 同步     | IPC        | 锁       |
| 内存     | 多         | 少       |
| 切换     | 复杂       | 简单     |
| CPU利用  | 低         | 高       |
| 生命周期 | 复杂，慢   | 简单，快 |
| 编程调试 | 简单       | 复杂     |
| 可靠性   | 不影响     | 影响     |
| 分布式   | 多核，多机 | 多核     |
