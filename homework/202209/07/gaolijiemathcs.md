### HTTPS和HTTP的区别？

|              | http                                                     | https                                                 |
| ------------ | -------------------------------------------------------- | ----------------------------------------------------- |
| 名称         | Hypertext Transfer protocol 超文本传输协议               | HyperText Transfer Protocol Secure 超文本传输安全协议 |
| 信息是否明文 | http信息明文传输                                         | https通过tls/ssl加密传输协议                          |
| 联系         | 不安全明文传输，未经安全加密协议，传输过程中容易被攻击。 | 可以视为HTTP+TLS/SSL协议组成。                        |
| 地址栏       | http://开头                                              | https://开头                                          |
| 默认端口     | 80                                                       | 443                                                   |
